# Flujo recomendado de práctica

1. Lee completamente la prueba o competencia  
2. Analiza restricciones y objetivos  
3. Divide el problema en partes pequeñas  
4. Implementa una solución funcional  
5. Optimiza rendimiento y estructura  
6. Revisa posibles mejoras  

> [!TIP]
> Resolver primero una versión simple y luego optimizar suele ser la mejor estrategia.
